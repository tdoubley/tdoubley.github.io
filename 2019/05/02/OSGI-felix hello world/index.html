OSGI-felix hello world

环境准备

  1. Eclipse IDE

    eclipse-jee-2019-03-R-win32-x86_64.zip

  2. felix

    http://mirrors.tuna.tsinghua.edu.cn/apache//felix/org.apache.felix.main.distribution-6.0.2.zip

新建工程

  1. 新建一个Java工程

  2. 将felix压缩包org.apache.felix.main.distribution-6.0.2.zip 中的conf、bundle以及bin (注意要将bin文件夹改名成lib避免与原工程文件夹重名)

  3. 添加Build Path

    将lib中的felix.jar添加到工程中

  4. 配置Run Configuration

​ 设置好后，点击Apply 然后点击Close即可

  1. 观察运行效果

  1. 错误原因分析

    • No config.properties found. : 这个是由于没有将felix中的conf文件夹复制到工程根目录

    • 错误: 找不到或无法加载主类 org.apache.felix.main.Main: 这个原因是运行时找不到相应的jar，这个可能时在修改Run Configuration没有将felix.jar添加到Build Path中,可以修改(只要按照步骤操作不会出现此错误)
